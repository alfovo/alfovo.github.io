The Design Challenge

Link Explorer is a free web tool that allows marketers to examine how their intelligent links are behaving and routing. Genius LInk is hoping that it will help their customers save time when testing how their intelligent links are behaving as they're being routed to various destinations around the world.

Key Features

  1. Show how an intelligent link reacts in different scenarios.
  2. Generate a tangible report that users can easily share.
  3. Create a tool that is fun to use and will become the go-to tool for marketers to checking their links.
